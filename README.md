# ATHENA-BT: A COMPLETELY OPEN-SOURCE BACKTESTING ENGINE :chart:
This is the official repository for AthenaBT, an in-house backtesting engine application developed by students at Quant Club, IIT Kharagpur. 

Hi, I am Shuvraneel Mitra, the official maintainer of this repository. At this point in time, the repository contains only a short demo of how to run a Python file in C++, using `pybindll`.
